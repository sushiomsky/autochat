## Privacy Practices

### Data Usage
- **No data collected**: Extension does not collect any user data
- **No data shared**: Extension does not share data with third parties
- **Local storage only**: All settings stored locally in browser

### Permissions Justification

1. **storage**: Save user settings and message lists
2. **activeTab**: Access current tab to send messages
3. **scripting**: Inject content scripts for automation
4. **alarms**: Schedule messages at specific intervals
5. **host_permissions (<all_urls>)**: Work on any chat website

## Review Notes for Google

This extension automates sending messages on chat websites with the user's permission and configuration. Key points:

1. **User Control**: All automation is explicitly configured by the user
2. **No Spam**: Includes daily limits and active hours to prevent abuse
3. **Privacy First**: No data collection, all storage is local
4. **Transparent**: Open source code available for review
5. **Educational**: Includes documentation and help system
6. **Safe**: No malicious code, no external connections except user-configured webhooks

The extension is designed for legitimate use cases like community management and does not facilitate spam or abuse.
